# Tutorial Troubleshooting Sales Aplikasi MODIS

## Jenis Sales

Aplikasi MODIS memiliki 2 jenis sales dengan karakteristik berbeda:

### 1. Sales Luar Kota
- Frekuensi: 1x keluar surat jalan dalam seminggu
- Periode: Senin sampai Sabtu

### 2. Sales Dalam Kota
- Frekuensi: 1 surat jalan per hari
- Periode: Senin sampai Sabtu

## Masalah Umum: Lupa Awali/Akhiri

Sales sering lupa melakukan **Awali** atau **Akhiri** pada aplikasi MODIS. Langkah pertama adalah menanyakan kepada sales:
- Di hari apa mereka lupa?
- Cek tanggal kejadian

## Solusi Troubleshooting

### A. Jika Lupa AWALI

1. Buka **Pengaturan** di aplikasi MODIS
2. Ubah tanggal secara manual ke tanggal kejadian
3. Set jam **di atas jam 16.00**
4. Kill aplikasi MODIS (tutup paksa)
5. Login kembali ke aplikasi
6. Kembalikan HP ke sales untuk melakukan **Awali** sendiri
7. Setelah selesai, sales akan mengembalikan HP
8. Set kembali tanggal dan jam ke **otomatis** (tanggal dan jam hari ini)
9. Selesai

### B. Jika Lupa AKHIRI

1. Buka **Pengaturan** di aplikasi MODIS
2. Ubah tanggal secara manual ke tanggal kejadian
3. Set jam **sebelum jam 08.00**
4. Kill aplikasi MODIS (tutup paksa)
5. Login kembali ke aplikasi
6. Kembalikan HP ke sales untuk melakukan **Akhiri** sendiri
7. Setelah selesai, sales akan mengembalikan HP
8. Set kembali tanggal dan jam ke **otomatis** (tanggal dan jam hari ini)
9. Selesai

## Kondisi Bermasalah (Awali/Akhiri Tidak Berhasil)

### Sales Luar Kota

Dilakukan pada **hari Senin**:

1. Buka aplikasi MODIS
2. Klik role **Admin**
3. Masuk ke **Pengaturan**
4. **Hapus data**
5. **Download kembali** data
6. Lakukan solusi troubleshooting seperti di atas (sesuai masalah Awali/Akhiri)

### Sales Dalam Kota

Dilakukan **setiap hari** dalam seminggu (Senin-Sabtu):

1. Buka aplikasi MODIS
2. Klik role **Admin**
3. Masuk ke **Pengaturan**
4. **Hapus data**
5. **Download kembali** data
6. Atasi masalah Awali/Akhiri sesuai prosedur
7. **Ulangi proses** hapus data dan download lagi untuk setiap hari yang bermasalah dalam 1 minggu

**Catatan:** Untuk sales dalam kota, proses ini harus diulang setiap hari karena mereka mengeluarkan 1 surat jalan per hari.

## Ringkasan Langkah Cepat

| Masalah | Set Jam | Aksi |
|---------|---------|------|
| Lupa Awali | Di atas 16.00 | Ubah tanggal → Kill app → Login → Sales awali → Reset otomatis |
| Lupa Akhiri | Sebelum 08.00 | Ubah tanggal → Kill app → Login → Sales akhiri → Reset otomatis |

**Tips:** Selalu tanyakan hari dan tanggal kejadian sebelum melakukan troubleshooting!
